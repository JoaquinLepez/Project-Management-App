Este proyecto es para la materia de Desarollo de Software.

La aplicación consiste en un gestor de proyectos en grupos en el cual será posible crear tareas, asignarlas a diferentes miembros de un equipo, asignarles fechas límite, etc.

Para ello se creó la siguiente historia de usuario para tener de referencia: 


**Historia de Usuario:**

Como **usuario individual o miembro de un equipo**, quiero **una plataforma integral para gestionar mis tareas diarias y proyectos** para aumentar mi productividad y organización.

**Criterios de Aceptación:**

1. Como usuario, quiero poder **registrarme y autenticarme en la plataforma** para acceder a mis tareas y proyectos de forma segura.

2. Como usuario, quiero poder **crear nuevas tareas** especificando un título, descripción, fecha de vencimiento, prioridad y categoría.

3. Como usuario, quiero poder **editar y actualizar las tareas existentes** para reflejar cambios en los detalles o el estado de la tarea.

4. Como usuario, quiero poder **eliminar tareas que ya no son relevantes o necesarias** para mantener mi lista de tareas limpia y organizada.

5. Como usuario, quiero poder **crear proyectos** y **agregar tareas a cada proyecto**, especificando hitos y fechas de entrega.

6. Como usuario, quiero poder **asignar tareas a diferentes miembros del equipo** si trabajo en un entorno colaborativo.

7. Como usuario, quiero tener **diferentes vistas para visualizar mis tareas y proyectos**, incluyendo vistas de lista, tablero o calendario.

8. Como usuario, quiero poder **colaborar en tiempo real con otros miembros del equipo**, permitiéndoles ver, editar y comentar tareas y proyectos compartidos.

9. Como usuario, quiero **integrar la aplicación con mi calendario externo** para sincronizar automáticamente las fechas de vencimiento y entregas de las tareas y proyectos.

10. Como usuario, quiero recibir **recordatorios y notificaciones** para tareas y proyectos próximos a su fecha de vencimiento.

11. Como usuario, quiero poder realizar un **seguimiento del progreso de mis tareas y proyectos**, incluyendo el tiempo dedicado a cada tarea y los hitos alcanzados.

12. Como usuario, quiero tener acceso a **análisis y métricas sobre mi productividad**, como el tiempo promedio dedicado a cada tarea y el número de tareas completadas por período de tiempo.

13. Como usuario, quiero que mis datos estén protegidos con **medidas de seguridad robustas**, como cifrado de datos y autenticación de dos factores.

Esta historia de usuario describe las necesidades y expectativas de los usuarios para la aplicación de Gestión de Tareas y Proyectos, estableciendo los criterios de aceptación para cada funcionalidad clave.
